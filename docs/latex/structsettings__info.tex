\doxysubsection{Detailed Description}
Structure in which we save the settings read from the settings file. ~\newline

\begin{DoxyItemize}
\item dataset\+\_\+file\+: Name of the file containing the dataset.
\item x\+\_\+dataset\+\_\+len\+: Size of the input variables.
\item x\+\_\+dataset\+\_\+len\+: Size of the output variables.
\item normalize\+: Whether the dataset must be normalized or not (true/false).
\item weights\+\_\+file\+: Name of the file containing the structure of the neural network and its weights/biases.
\item Si\+M\+E\+C\+\_\+output\+\_\+file\+: Name of the file in which the Si\+M\+EC algorithms shall write their outputs.
\item n\+\_\+iterations\+: Number of iterations.
\item delta\+: The integration step delta of the Si\+M\+E\+C-\/1D algorithm.
\item invert\+\_\+direction\+: Whether to invert the direction of Si\+M\+E\+C-\/1D or not (true/false).
\item starting\+\_\+point\+\_\+file\+: The file containing the point for which we build the equivalence class.
\item normalize\+\_\+starting\+: If Si\+M\+E\+C-\/1D or Si\+M\+Exp-\/1D are selected, this parameter specifies whethere the dataset must be normalized or not (true/false). If true, the maximum and the minimum of the dataset the starting point comes from must be provided in a \mbox{\hyperlink{structmax__mix__struct}{max\+\_\+mix\+\_\+struct}}.
\item algo\+: Algorithm to run\+: Si\+M\+E\+C-\/1D / Si\+M\+Exp-\/1D / Predict.
\item epsilon\+: Epsilon of Si\+M\+Exp-\/1D algorithm.
\item delta\+\_\+simexp\+: Delta of the Si\+M\+Exp-\/1D algorithm, namely the maximum distance from the starting point.
